# Meteo 473 Lake Effect Snow Potential Index (LESPI) — Group 5

## Project Description
The Lake Effect Snow Potential Index (LESPI) is a specialized diagnostic tool designed to quantify the risk and intensity of lake-effect snow (LES) events. For this project, we developed a case-study demonstration using high-resolution Global Forecast System (GFS) data from a significant 2022 event. By processing thermodynamic and moisture variables, LESPI transforms complex model output into a single, intuitive percentage (0–100%) that highlights the potential for high-impact snow bands.
